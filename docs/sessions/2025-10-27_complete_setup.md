**Problem:** Slovak characters displayed as `?` in Windows console  
**Example:** `?no` instead of `√Åno`

**Solution:**
```python
# Force UTF-8 output for Windows console
if sys.platform == 'win32':
    os.system('chcp 65001 > nul')
    sys.stdout = io.TextIOWrapper(
        sys.stdout.buffer,
        encoding='utf-8',
        errors='replace'
    )
```
